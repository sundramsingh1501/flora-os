# Flora OS — Personal AI Executive Assistant

A full-stack personal productivity dashboard built with Streamlit.

**Features:**
- Morning Brief with AI-ranked top stories
- Gmail integration — job emails (interviews, assessments, rejections)
- Job Board with AI fit scoring across LinkedIn, Indeed, Adzuna, RemoteOK
- Live market watchlist (stocks + crypto)
- Daily email delivery at your configured time

## Setup

Copy `.env.example` to `.env` and fill in your API keys.
